# Backend — Node.js + Express + TypeScript

> Scaffolding coming soon. Run `npm run start:be` from root.

## Stack

- **Runtime**: Node.js 18+
- **Framework**: Express.js
- **Language**: TypeScript
- **Database**: MongoDB Atlas (Mongoose ODM)
- **Auth**: JWT + Passport.js
- **Payment**: Stripe SDK
- **Deployment**: Render.com (free tier)
